#说明
CNKI批量下载，2025年5月亲测可用，在知网官网、淘bao和pdd的盗版知网上都可以用。
插件生效后，会出现一个大红色的批量下载，一点就会下载当前页面的所有论文
需要添加其他知网源的（包括各大高校的图书馆资源），请在// @match 后面添加自己用的知网的网址！
在每篇文献间添加了15s的等待时间，防止封IP和弹验证窗口。

#注意
允许多弹窗（第一次运行，网页右上角弹出来那个图标），并关闭每次下载都要询问
